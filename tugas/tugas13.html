Tugas E-Learning – DIV & SPAN

1. Apa bedanya DIV dan SPAN?
DIV adalah elemen block-level yang digunakan untuk mengelompokkan bagian besar pada halaman (misalnya header, konten, footer). DIV selalu mulai dari baris baru dan lebarnya mengikuti lebar halaman.

SPAN adalah elemen inline yang dipakai untuk memberi style pada teks atau bagian kecil di dalam sebuah kalimat tanpa memulai baris baru, misalnya mewarnai satu kata saja.

2. Contoh hasil tampilan website (codingan):

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Explore Indonesia</title>

    <style>
        body {
            font-family: Arial, sans-serif;
            background: #f5f7fa;
            margin: 0;
            padding: 0;
        }

        /* DIV sebagai pembungkus besar (block-level) */
        .header {
            background: #0a58ca;
            color: white;
            padding: 20px;
            text-align: center;
        }

        .menu {
            margin-top: 10px;
        }

        /* SPAN untuk item menu (inline) */
        .menu span {
            background: white;
            color: #0a58ca;
            padding: 6px 12px;
            border-radius: 4px;
            margin: 0 4px;
            cursor: pointer;
            font-weight: bold;
        }

        .container {
            background: white;
            width: 80%;
            margin: 25px auto;
            padding: 20px;
            border-radius: 10px;
            box-shadow: 0 3px 10px rgba(0,0,0,0.1);
        }

        .content {
            display: flex;
            gap: 20px;
            margin-top: 20px;
        }

        .text-box {
            flex: 1;
        }

        .image-box img {
            width: 260px;
            border-radius: 8px;
        }

        .list-section {
            background: #eef3ff;
            padding: 12px;
            border-radius: 8px;
            margin-top: 20px;
        }
    </style>
</head>

<body>

    <div class="header">
        <h2>Jelajahi Keindahan Indonesia</h2>
        <div class="menu">
            <span>Home</span>
            <span>Destinasi</span>
            <span>Artikel</span>
            <span>Kontak</span>
        </div>
    </div>

    <div class="container">
        <div class="content">
            <div class="text-box">
                <h3>Wisata Alam</h3>
                <p>Temukan tempat-tempat indah di seluruh Indonesia mulai dari gunung, pantai, sampai air terjun.</p>
                <button>Lihat Selengkapnya</button>
            </div>

            <div class="image-box">
                <img src="https://via.placeholder.com/260x170?text=Indonesia" alt="Gambar wisata">
            </div>
        </div>

        <div class="list-section">
            <b>Destinasi Populer:</b>
            <ul>
                <li>Bromo – Jawa Timur</li>
                <li>Raja Ampat – Papua</li>
                <li>Pantai Kuta – Bali</li>
            </ul>
        </div>
    </div>

</body>
</html>
