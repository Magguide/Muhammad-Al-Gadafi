Tugas Dasar CSS

1. Apa kegunaan CSS?
CSS digunakan untuk mengatur tampilan halaman web seperti warna, ukuran tulisan, jarak antarelemen, posisi layout, hingga efek visual. Dengan CSS, halaman lebih rapi, konsisten, dan mudah disesuaikan tanpa mengubah struktur HTML.

2. Buatlah contoh CSS Embedded berdasarkan struktur web Anda!
Berikut contoh penulisan CSS secara embedded menggunakan tag <style> di dalam <head>. Contoh di bawah ini hanya ilustrasi, tinggal sesuaikan selector-nya dengan struktur website Anda.

<style> /* Reset sederhana */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Mengatur tampilan dasar halaman */ body { font-family: Arial, sans-serif; font-size: 16px; background: #f4f6f9; color: #222; padding: 20px; } /* Container utama */ .container { max-width: 900px; margin: auto; background: white; padding: 25px; border-radius: 8px; box-shadow: 0 0 10px rgba(0,0,0,0.1); } /* Judul */ header h1 { font-size: 26px; color: #1e4fa3; margin-bottom: 15px; } /* Navigasi */ nav ul { list-style: none; display: flex; gap: 15px; } nav a { text-decoration: none; color: #1e4fa3; font-weight: bold; } /* Artikel */ .article { padding: 12px; border-left: 4px solid #1e4fa3; background: #eef4ff; margin-top: 15px; } </style>
